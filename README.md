# Weather Dashboard

## What This Dashboard Shows
This weather dashboard displays current weather conditions and a 3-day forecast for Boston using real-time data from the Open-Meteo API.

## API Used
Open-Meteo API (no API key required)  
https://open-meteo.com/


## Features
- Current temperature, humidity, wind speed, and weather description
- 3-day forecast with high and low temperatures
- Weather icons using emoji
- Toggle between Celsius and Fahrenheit
- Loading and error handling


## What I Learned
- How to fetch real-time data using an API
- How to read and navigate JSON responses
- How to map numerical weather codes to readable descriptions
- How to dynamically update the DOM with API data
- How to handle loading states and errors
- The importance of user interaction in making a tool feel complete
